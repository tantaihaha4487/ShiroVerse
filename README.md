# ShiroVerse

Library plugin for ShiroVerse project.

# Installation

How to add ShiroCore library to your project.

Add to pom.xml

```
	<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>
```

```
    <dependency>
	    <groupId>com.github.tantaihaha4487</groupId>
	    <artifactId>ShiroCore</artifactId>
	    <version>1.21-SNAPSHOT</version>
	</dependency
```
